En este repositório mantengo mi portafolio de Ciencia de Datos.
Son tres ficheros del tipo 'Jupyter notebook' cada uno con algunos modelos distintios del mismo tipo.
Por ejemplo, en el fichero MachineLearning-Clasificación.ipynb hay cuatro modelos distintos de clasificación:
-Decision Tree
-K-Nearest Neighbors
-Logistic Regression
-Support Machine Vector

El fichero de nombre port.py es el fichero de python utilizado para generar el dashboard que se puede ver en el
enlace https://nico-port.herokuapp.com/ construído en python con la biblioteca 'dash'.



  --  English  --

In this repository I keep my Data Science portfolio.
There are three Jupiter notebook files, each one with some different models of the same kind.
For example, in the file MachineLearning-Clasificación.ipynb there are four distinct classification models:
-Decision Tree
-K-Nearest Neighbors
-Logistic Regression
-Support Machine Vector

The file called port.py is the one used to generate the dashboard that can be seen at the link
https://nico-port.herokuapp.com/ built with python using the library 'dash'
